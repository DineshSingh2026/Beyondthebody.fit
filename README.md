# Beyond The Body

## Environment Switching

Use these commands from the project root:

- `npm run env:local` - switch `.env` to local PostgreSQL (`.env.local`)
- `npm run env:render` - switch `.env` to Render PostgreSQL (`.env.render`)
- `npm run env:show` - print the active DB profile and host from `.env`

After switching, start the app normally:

- `npm start`
